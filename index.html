Kanwise
Kanban Board

A Kanban application in Angular is a project management tool that allows users to visualize and track the progress of work items through different stages of development. It is based on the Kanban method, which originated in the manufacturing industry and has since been adopted by software development teams to improve workflow efficiency and productivity.

In an Angular Kanban application, users can create a board with different columns representing the different stages of work (e.g. "To Do", "In Progress", "Done"). They can then add cards to the board, representing individual work items, and drag and drop them between columns to reflect their progress.

Tools & Technologies used are -:
Frontend- HTML5, CSS, BOOTSTRAP, Angular.
Backend- Microservices, Springboot
Databases – MySQL and MongoDB database.
